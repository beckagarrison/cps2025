================================================================================
                             TROUBLESHOOTING
================================================================================

Problem: "Build failed"
Solution: Run `npm run build` locally to see the error

Problem: "Environment variables not found"
Solution: Make sure they start with VITE_ and redeploy after adding

Problem: "Domain not connecting"
Solution: Wait for DNS propagation (up to 48 hours) and verify DNS records

Problem: "Login not working"
Solution: Check Supabase Site URL and Redirect URLs are correct

Problem: "404 on page refresh"
Solution: Already fixed in vercel.json (SPA routing enabled)
