## ğŸ“‹ How to Use

### Creating Your First Case

1. **Navigate to "My Cases" tab** (first tab in navigation)
2. **Click "Create New Case"** button
3. **Complete the 4-step wizard**:
   - **Step 1**: Basic information (case name, docket number, county, status, date opened)
   - **Step 2**: Case worker and court details
   - **Step 3**: Add children and important dates
   - **Step 4**: Review and add notes

4. **Click "Create Case"** to save

### Selecting a Case

- Use the **Case Selector dropdown** (located below the state selector)
- Click on any case card in **"My Cases"** dashboard
- The active case is always displayed in the selector

### Editing a Case

1. Go to **"My Cases"** tab
2. Click the **3-dot menu** on any case card
3. Select **"Edit Case"**
4. Update information in the wizard
5. Click **"Save Changes"**

### Deleting a Case

1. Go to **"My Cases"** tab
2. Click the **3-dot menu** on the case you want to delete
3. Select **"Delete Case"**
4. Confirm deletion

âš ï¸ **Warning**: Deleting a case will remove all associated data (documents, timeline, violations, etc.). This cannot be undone.
