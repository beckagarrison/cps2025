## ‚ú® ANIMATIONS:

### **Custom Keyframe Animations:**

**1. Fade-In:**
```css
@keyframes fade-in {
  from { opacity: 0; }
  to { opacity: 1; }
}
```
- Duration: 1s
- Applied to: Main container

**2. Slide-Up:**
```css
@keyframes slide-up {
  from { 
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
- Duration: 0.8s
- Applied to: All major sections
- Staggered delays: 0.2s, 0.4s, 0.6s, etc.

**3. Float:**
```css
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-20px); }
}
```
- Duration: 6s infinite
- Applied to: Logo
- Creates smooth floating effect

### **Tailwind Animations:**
- **Pulse:** Background circles (3s, 4s, 5s durations)
- **Scale:** Hover effects on cards
- **Translate:** Arrow slide on hover
- **All transitions:** Smooth with transition-all
