# PDF Text Extraction Guide

## How to Copy Text from Your CPS PDF Documents

### ‚úÖ METHOD 1: Direct Copy/Paste (Recommended - Takes 30 seconds)

1. **Open your PDF file** in any PDF reader (Adobe, Chrome, Preview, etc.)

2. **Select ALL text:**
   - **Windows:** Press `Ctrl + A`
   - **Mac:** Press `Cmd + A`
   - You should see all the text highlighted in blue

3. **Copy the text:**
   - **Windows:** Press `Ctrl + C`
   - **Mac:** Press `Cmd + C`

4. **Go back to the CPS app** and click "Upload File from Computer"

5. **Select your PDF file** - The app will:
   - ‚úÖ Automatically fill in the document title
   - ‚úÖ Show you instructions
   - ‚úÖ Keep the form ready for pasting

6. **Click in the text box** and paste:
   - **Windows:** Press `Ctrl + V`
   - **Mac:** Press `Cmd + V`

7. **Click "Add & Analyze Document"** - Done! üéâ

---

### ‚úÖ METHOD 2: Using Google Drive (If Method 1 Doesn't Work)

Some PDFs are scanned images or have text protection. If you can't copy text directly:

1. **Upload PDF to Google Drive:**
   - Go to [drive.google.com](https://drive.google.com)
   - Click "New" ‚Üí "File upload"
   - Select your PDF

2. **Convert to Google Docs:**
   - Right-click the PDF in Google Drive
   - Select "Open with" ‚Üí "Google Docs"
   - Google will automatically convert it to text (takes 5-10 seconds)

3. **Copy the text:**
   - Press `Ctrl + A` (Windows) or `Cmd + A` (Mac) to select all
   - Press `Ctrl + C` (Windows) or `Cmd + C` (Mac) to copy

4. **Return to CPS app:**
   - Upload the PDF file to auto-fill the title
   - Paste the text in the box
   - Click "Add & Analyze Document"

---

### ‚úÖ METHOD 3: Online OCR Tools (For Scanned Documents)

If the PDF is a scanned image (like a photo of a document):

**Free OCR Tools:**
- **[Adobe Online OCR](https://www.adobe.com/acrobat/online/pdf-to-text.html)** - No account needed
- **[OnlineOCR.net](https://www.onlineocr.net/)** - Free, no registration
- **[PDF2Go](https://www.pdf2go.com/ocr-pdf)** - Simple interface

**Steps:**
1. Go to any OCR tool above
2. Upload your PDF
3. Wait for conversion (usually 10-30 seconds)
4. Download or copy the extracted text
5. Paste into CPS app

---

### ‚úÖ METHOD 4: Using Windows/Mac Built-in Tools

**Windows 10/11 with Microsoft OneNote:**
1. Open OneNote (free with Windows)
2. Insert ‚Üí Pictures ‚Üí select your PDF
3. Right-click the image ‚Üí "Copy Text from Picture"
4. Paste the text into CPS app

**Mac with Preview:**
1. Open PDF in Preview
2. Tools ‚Üí Text Selection tool
3. Select all text (Cmd + A)
4. Copy (Cmd + C)
5. Paste into CPS app

## ‚ö†Ô∏è Common Issues & Solutions

### "The text is all garbled or has weird characters"
**Solution:** Your PDF might be corrupted or encrypted. Try:
- Opening in a different PDF reader
- Using Google Drive method (Method 2)
- Using an OCR tool (Method 3)

### "I can't select any text in my PDF"
**Solution:** Your PDF is likely a scanned image. Use:
- Method 2 (Google Drive) or
- Method 3 (OCR tools)

### "The file is too large to copy all at once"
**Solution:** Copy in sections:
1. Select first half of document (don't use Ctrl+A)
2. Copy and paste into CPS app
3. Add more text to the same text box
4. Repeat until complete

### "Some pages are missing text"
**Solution:** Your PDF might have mixed content (some text, some images):
- Copy what you can using Method 1
- Use OCR for image pages using Method 3
- Manually type any critical missing information
